# PlantCare AI 🌱

PlantCare AI is a machine learning project that detects plant diseases using Transfer Learning.

## Dataset
New Plant Diseases Dataset (87,000 images, 38 classes)

## Technologies Used
- Python
- TensorFlow / Keras
- MobileNetV2
- Flask
- OpenCV

## Features
- Plant disease classification
- Image preprocessing & augmentation
- Transfer learning using MobileNetV2
- Web application for disease detection
