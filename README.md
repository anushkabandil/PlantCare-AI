## 🌱 Why PlantCare AI?

Plant diseases silently damage crops, reduce productivity, and impact farmers globally.
**PlantCare AI** leverages the power of Deep Learning to provide **fast, accurate, and scalable disease detection** using just a leaf image.

---

## 🔥 Key Highlights

* ⚡ Instant disease prediction
* 🧠 Transfer Learning using MobileNetV2
* 🌍 Real-world agricultural application
* 🌐 Interactive web-based interface
* 🎯 Supports 38 plant disease classes

---

## 📊 Dataset Overview

* 📸 **87,000+ images**
* 🌿 **38 disease categories**
* ✅ Includes both healthy & diseased leaves

## 🧠 Model Insights

* Utilizes **MobileNetV2** pre-trained on ImageNet
* Fine-tuned on plant disease dataset
* Applies **transfer learning** for better generalization
* Optimized for:

  * ⚡ Speed (lightweight model)
  * 🎯 Accuracy (multi-class classification)
* Handles diverse plant species and disease patterns
